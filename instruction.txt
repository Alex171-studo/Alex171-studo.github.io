## ARCHITECTURE CIBLE À CRÉER

### _config.yml
```yaml
title: "Alex171 | Security Researcher"
description: "HTB Writeups & Cybersecurity Journey"
baseurl: ""
url: "https://Alex171-studo.github.io"
author:
  name: "Alex171"
  email: ""
  github: "Alex171-studo"
  htb: ""        # ton pseudo HTB ici
  tryhackme: ""  # si tu en as un

theme: minima   # on va override avec notre propre layout

plugins:
  - jekyll-feed
  - jekyll-seo-tag

defaults:
  - scope:
      path: "_posts"
      type: "posts"
    values:
      layout: "post"
      author: "Alex171"
```


## DESIGN SYSTEM À APPLIQUER (tout dans un seul style.css)

Palette :
- Background : #0a0a0a (noir profond)
- Surface : #111111 / #1a1a1a (cards, panels)
- Accent primaire : #00ff41 (vert terminal)
- Accent secondaire : #ff6b35 (orange)
- Text principal : #e0e0e0
- Text secondaire : #888888
- Borders : #222222

Typographie :
- Monospace pour les titres et code : 'JetBrains Mono' ou 'Fira Code' (Google Fonts)
- Sans-serif pour le corps : 'Inter'

Composants à définir une seule fois :
- .card (writeup cards)
- .badge-difficulty (Easy=vert, Medium=orange, Hard=rouge, Insane=violet)
- .badge-platform
- .skill-bar
- .terminal-block (pour les blocs de code dans les writeups)
- .btn
